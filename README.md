# Email Spam Detection

Machine learning project to detect spam messages using two models.

## Models Used
- Naive Bayes
- Decision Tree

## Dataset
SMS Spam Collection — 5,572 messages (Kaggle/UCI)

## Libraries
pandas, scikit-learn, nltk, matplotlib, seaborn

## Results
| Model         | Accuracy |
|---------------|----------|
| Naive Bayes   | ~97%     |
| Decision Tree | ~95%     |
